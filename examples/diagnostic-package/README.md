# Example Diagnostic Package

This directory documents the expected shape of a Doctor link diagnostic package.

It is intentionally lightweight and does not contain real logs, secrets, or user data.

## Example files

- `doctor-report.json`: machine-readable package summary.
- `summary.md`: human-readable package summary.
- `verification-result.json`: sample verification result.
- `ai-task.md`: sample AI task handoff content.

Use this example as a reference for integrations and documentation.
