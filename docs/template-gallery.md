# Template Gallery

Doctor link examples and templates are stored under `examples/`.

## Sample project

Path:

```text
examples/sample-project/
```

Includes:

- `.doctorlink/diagnosis.yml`
- `.doctorlink/reproduce.yml`
- `.doctorlink/test-matrix.yml`

Use this as a starting point for project-level Doctor link configuration.

## Example diagnostic package

Path:

```text
examples/diagnostic-package/
```

Includes lightweight example package files without real user data.

## Example AI handoff

Path:

```text
examples/ai-handoff/
```

Shows the expected shape of an AI Coding handoff package.

## Example before / after workflow

Path:

```text
examples/before-after/
```

Shows the before / after workflow commands and expected outputs.

## Safety note

Examples must not contain real secrets, credentials, personal data, private logs, or customer data.
