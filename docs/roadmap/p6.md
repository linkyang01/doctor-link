# Doctor link P6 Roadmap

P6 turns Doctor link from a productized diagnostic tool into a diagnostic protocol standard and ecosystem platform for AI Coding workflows.

P6 has completed P6.1 schema standardization and P6.2 local compatibility / conformance validation. P6.3 and later remain future work unless separately authorized.

Current boundary:

- P6.1 Diagnostic Package Schema v1: completed.
- P6.2 Compatibility and conformance test suite: completed.
- P6.3+ SDK, plugin, integration, signing, privacy, governance, knowledge base, and public ecosystem work: not started.
- No Web platform, cloud service, account system, marketplace, PyPI publishing, release tag, or GitHub Release is authorized by P6.2.
