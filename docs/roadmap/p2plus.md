# P2+ Roadmap

P2+ is a mainline enhancement task group after P2.

Focus areas:

1. Evidence review.
2. Verification workbench.
3. AI task handoff.
4. Web UI usability.
5. Fixtures and validation.
6. Documentation and audit.

P2+ keeps the local workbench read-only by default.

Completion requires checked TODO items, passing tests, passing CI, and a final audit.
