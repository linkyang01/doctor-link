# Doctor link v0.1.1 Release Notes

Release date: 2026-06-29

## Summary

`v0.1.1` is a post-release polish release after the first public GitHub Release (`v0.1.0-rc.1`). It aligns project documentation with the published release status and improves the guided `wizard` workflow for scripted and multi-tool AI handoff.

## Highlights

- `doctor-link wizard --tool` selects an AI handoff profile without interactive prompts
- `doctor-link wizard --handoff/--no-handoff` and `--collect-evidence/--no-collect-evidence` for non-interactive runs
- Project status, validation certificates, and README files reflect the published GitHub Release
- P8 no-code UX roadmap marked largely complete

## Install

### From GitHub Release wheel

```bash
pip install https://github.com/linkyang01/doctor-link/releases/download/v0.1.1/doctor_link-0.1.1-py3-none-any.whl
doctor-link --version
```

### From source

```bash
git clone https://github.com/linkyang01/doctor-link.git
cd doctor-link
git checkout v0.1.1
pip install -e .
doctor-link --version
```

## Quick commands

```bash
doctor-link wizard --folder . --summary "startup issue" --tool codex --handoff --json
doctor-link diagnose-now . --full --handoff
doctor-link home
```
