# Doctor link v0.1.2 Draft Release Notes

Release date: TBD — publication requires explicit authorization

## Summary

`v0.1.2` expands AI handoff tooling with three new profiles and adds guided-workflow CLI polish for scripted diagnosis and handoff generation.

## Highlights

- New handoff profiles: `grok`, `windsurf`, `cline`
- `doctor-link diagnose-now --tool` aligned with `wizard --tool`
- `doctor-link handoff list` and `doctor-link handoff check`
- `doctor-link handoff --json` for scripted automation
- Default guided handoff tool unified to `cursor`
- Documentation aligned to the real release state and kept in draft until publication

## Quick commands

```bash
doctor-link handoff list
doctor-link wizard --folder . --summary "startup issue" --tool grok --handoff --json
doctor-link diagnose-now . --handoff --tool grok --json
doctor-link handoff check <package_dir> --tool cursor --json
doctor-link handoff <package_dir> --tool codex --json
```
