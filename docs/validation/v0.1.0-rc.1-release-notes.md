# Doctor link v0.1.0-rc.1 Release Notes

Release date: 2026-06-29

## Summary

`v0.1.0-rc.1` is the first public release candidate for Doctor link as a local-first diagnostic CLI and AI collaboration tool.

## Highlights

- Guided workflows: `wizard`, `diagnose-now --full`, `home`
- Developer-grade pipeline: evidence, verification, workbench, handoff
- Cloud CI certified on Python 3.10 / 3.11 / 3.12
- Local Mac validation recorded on 2026-06-29

## Install

```bash
git clone https://github.com/linkyang01/doctor-link.git
cd doctor-link
pip install -e .
doctor-link --version
```

## Quick commands

```bash
doctor-link wizard
doctor-link diagnose-now . --full --handoff
doctor-link report .
doctor-link verify <package_dir>
doctor-link handoff <package_dir> --tool cursor
```

## Boundaries

This release candidate does **not** include:

- hosted Web platform or cloud sync
- user accounts or telemetry
- PyPI publication (unless separately authorized)
- real signing key management
