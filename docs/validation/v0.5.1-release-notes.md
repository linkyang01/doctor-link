# v0.5.1 Release Notes

Version 0.5.1 is a distribution and documentation consistency patch for the v0.5 feature set. It does not change the diagnostic, reproduction, repair, or verification contract.

## Changed

- The PyPI long description now presents v0.5.1 as the current release instead of retaining pre-publication v0.4 text.
- PyPI and `pipx` are the primary public installation path; the immutable GitHub Release remains a fallback.
- English, Chinese, installation, project-status, release-policy, and validation documents consistently record the v0.5 feature set and PyPI availability.
- PyPI publication uses the dedicated Trusted Publishing workflow, protected `pypi` environment, short-lived OIDC credentials, and digital attestations.

## Required release evidence

- complete local and GitHub Actions validation;
- immutable v0.5.1 GitHub Release;
- PyPI Trusted Publishing with matching asset hashes;
- clean `pipx install doctor-link==0.5.1` validation.
