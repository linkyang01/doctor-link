# Doctor link v0.1.0-rc.1

Release date: 2026-06-29

## Summary

Cloud-CI-certified release candidate with guided diagnosis workflows, unified Vly adapter, and expanded examples.

## Install

### From GitHub tag

```bash
git clone https://github.com/linkyang01/doctor-link.git
cd doctor-link
git checkout v0.1.0-rc.1
pip install -e .
doctor-link --version
```

### From release wheel

Download `doctor_link-0.1.0rc1-*.whl` from the GitHub Release assets, then:

```bash
pip install doctor_link-0.1.0rc1-*.whl
```

### From PyPI (when enabled)

```bash
pip install doctor-link==0.1.0rc1
```

## Highlights since the first RC draft

- `doctor-link wizard`, `home`, and `diagnose-now --full`
- Friendly errors and `--version`
- `report` reads `.doctorlink/diagnosis.yml`
- Unified `VlyAdapter` registry
- Examples: `python-api-bug`, `media-playback-library`
- No-code quick start guides in English and Chinese
