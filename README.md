# Doctor link

**Language / 语言：** [English](README.en.md) | [中文](README.zh-CN.md)

---

Doctor link is a human-AI shared diagnostic layer for software projects.

Doctor link 是一个面向软件项目的人机协同诊断层。

It helps users, developers, and AI coding tools collect evidence, reproduce problems, preserve human-confirmed issues, generate AI-ready debugging tasks, and verify fixes.

它帮助用户、开发者和 AI 编程工具采集证据、复现问题、保留用户确认的问题、生成可交给 AI 的修复任务，并验证修复结果。

## Current status

Doctor link is in the foundation-building stage. The current priority is to stabilize the diagnostic protocol, standard diagnostic package, user assertion mechanism, problem map, AI task generation, and fix verification loop.

Doctor link 当前处于地基建设阶段。当前优先级是稳定诊断协议、标准诊断包、用户确认问题机制、问题地图、AI 任务生成和修复验证闭环。
